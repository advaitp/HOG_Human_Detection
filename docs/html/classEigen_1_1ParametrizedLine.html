Detailed Description

template<typename Scalar_, int _AmbientDim, int Options_>
class Eigen::ParametrizedLine< Scalar_, _AmbientDim, Options_ >

A parametrized line.

\geometry_module

A parametrized line is defined by an origin point and a unit direction vector such that the line corresponds to the set , .

Template Parameters
Scalar_the scalar type, i.e., the type of the coefficients
_AmbientDimthe dimension of the ambient space, can be a compile time value or Dynamic.
