Expression of a fixed-\/size or dynamic-\/size sub-\/vector. 


\begin{DoxyTemplParams}{Template Parameters}

This class represents an expression of either a fixed-\/size or dynamic-\/size sub-\/vector. It is the return type of Dense\+Base\+::segment(\+Index,\+Index) and Dense\+Base\+::segment$<$int$>$(\+Index) and most of the time this is the only way it is used.

However, if you want to directly manipulate sub-\/vector expressions, for instance if you want to write a function returning such an expression, you will need to use this class.
