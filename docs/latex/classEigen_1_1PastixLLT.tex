class Eigen\+::\+Pastix\+LLT$<$ Matrix\+Type\+\_\+, Up\+Lo\+\_\+ $>$}
A sparse direct supernodal Cholesky (\mbox{\hyperlink{classEigen_1_1LLT}{LLT}}) factorization and solver based on the Pa\+StiX library. 

This class is used to solve the linear systems A.\+X = B via a LL$^\wedge$T supernodal Cholesky factorization available in the Pa\+StiX library. The matrix A should be symmetric and positive definite WARNING Selfadjoint complex matrices are not supported in the current version of Pa\+StiX The vectors or matrices X and B can be either dense or sparse
