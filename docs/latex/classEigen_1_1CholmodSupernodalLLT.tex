\subsubsection*{template$<$typename Matrix\+Type\+\_\+, int Up\+Lo\+\_\+ = Lower$>$\newline
class Eigen\+::\+Cholmod\+Supernodal\+LLT$<$ Matrix\+Type\+\_\+, Up\+Lo\+\_\+ $>$}
A supernodal Cholesky (\mbox{\hyperlink{classEigen_1_1LLT}{LLT}}) factorization and solver based on Cholmod. 

This class allows to solve for A.\+X = B sparse linear problems via a supernodal LL$^\wedge$T Cholesky factorization using the Cholmod library. This supernodal variant performs best on dense enough problems, e.\+g., 3D FEM, or very high order 2D FEM. The sparse matrix A must be selfadjoint and positive definite. The vectors or matrices X and B can be either dense or sparse.
