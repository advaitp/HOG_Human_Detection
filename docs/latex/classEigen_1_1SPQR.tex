\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Matrix\+Type\+\_\+$>$\newline
class Eigen\+::\+SPQR$<$ Matrix\+Type\+\_\+ $>$}
\mbox{\hyperlink{structEigen_1_1Sparse}{Sparse}} QR factorization based on Suite\+Sparse\+QR library. 

This class is used to perform a multithreaded and multifrontal rank-\/revealing QR decomposition of sparse matrices. The result is then used to solve linear leasts\+\_\+square systems. Clearly, a QR factorization is returned such that A$\ast$P = Q$\ast$R where \+:

P is the column permutation. Use \mbox{\hyperlink{classEigen_1_1SPQR_af7a3ad7b3fa2814354159acea9e76d53}{cols\+Permutation()}} to get it.

Q is the orthogonal matrix represented as Householder reflectors. Use \mbox{\hyperlink{classEigen_1_1SPQR_a5bc3d5be4365720afd2aa0fb2ef9f78e}{matrix\+Q()}} to get an expression and \mbox{\hyperlink{classEigen_1_1SPQR_a5bc3d5be4365720afd2aa0fb2ef9f78e}{matrix\+Q()}}.transpose() to get the transpose. You can then apply it to a vector.

R is the sparse triangular factor. Use matrix\+QR() to get it as \mbox{\hyperlink{classEigen_1_1SparseMatrix}{Sparse\+Matrix}}. NOTE \+: The Index type of R is always Suite\+Sparse\+\_\+long. You can get it with SPQR\+::\+Index
