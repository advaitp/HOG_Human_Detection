\subsubsection*{template$<$typename Matrix\+Type\+\_\+, int Up\+Lo\+\_\+, typename Ordering\+\_\+$>$\newline
class Eigen\+::\+Simplicial\+LLT$<$ Matrix\+Type\+\_\+, Up\+Lo\+\_\+, Ordering\+\_\+ $>$}
A direct sparse \mbox{\hyperlink{classEigen_1_1LLT}{LLT}} Cholesky factorizations. 

This class provides a LL$^\wedge$T Cholesky factorizations of sparse matrices that are selfadjoint and positive definite. The factorization allows for solving A.\+X = B where X and B can be either dense or sparse.

In order to reduce the fill-\/in, a symmetric permutation P is applied prior to the factorization such that the factorized matrix is P A P$^\wedge$-\/1.
