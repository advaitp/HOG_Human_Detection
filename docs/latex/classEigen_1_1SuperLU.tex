\subsubsection*{template$<$typename Matrix\+Type\+\_\+$>$\newline
class Eigen\+::\+Super\+LU$<$ Matrix\+Type\+\_\+ $>$}
A sparse direct LU factorization and solver based on the \mbox{\hyperlink{classEigen_1_1SuperLU}{Super\+LU}} library. 

This class allows to solve for A.\+X = B sparse linear problems via a direct LU factorization using the \mbox{\hyperlink{classEigen_1_1SuperLU}{Super\+LU}} library. The sparse matrix A must be squared and invertible. The vectors or matrices X and B can be either dense or sparse.

\end{DoxyTemplParams}
\begin{DoxyWarning}{Warning}
This class is only for the 4.\+x versions of \mbox{\hyperlink{classEigen_1_1SuperLU}{Super\+LU}}. The 3.\+x and 5.\+x versions are not supported.
