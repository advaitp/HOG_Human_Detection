\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Matrix\+Type, int Up\+Lo\+\_\+$>$\newline
class Eigen\+::\+Pardiso\+LLT$<$ Matrix\+Type, Up\+Lo\+\_\+ $>$}
A sparse direct Cholesky (\mbox{\hyperlink{classEigen_1_1LLT}{LLT}}) factorization and solver based on the PARDISO library. 

This class allows to solve for A.\+X = B sparse linear problems via a LL$^\wedge$T Cholesky factorization using the Intel MKL PARDISO library. The sparse matrix A must be selfajoint and positive definite. The vectors or matrices X and B can be either dense or sparse.

By default, it runs in in-\/core mode. To enable PARDISO\textquotesingle{}s out-\/of-\/core feature, set\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{solver.pardisoParameterArray()[59] = 1; }

\end{DoxyCode}
