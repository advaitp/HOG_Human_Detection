\subsubsection*{template$<$typename Expression\+Type, template$<$ typename $>$ class Storage\+Base$>$\newline
class Eigen\+::\+No\+Alias$<$ Expression\+Type, Storage\+Base $>$}
Pseudo expression providing an operator = assuming no aliasing. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Expression\+Type} & the type of the object on which to do the lazy assignment\\
\hline
\end{DoxyTemplParams}
This class represents an expression with special assignment operators assuming no aliasing between the target expression and the source expression. More precisely it alloas to bypass the Eval\+Before\+Assign\+Bit flag of the source expression. It is the return type of \mbox{\hyperlink{classEigen_1_1MatrixBase_a7f15120bb9c88c30aa5d993bd6d138e1}{Matrix\+Base\+::noalias()}} and most of the time this is the only way it is used.
