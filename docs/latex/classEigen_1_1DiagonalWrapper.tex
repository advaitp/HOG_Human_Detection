class Eigen\+::\+Diagonal\+Wrapper$<$ Diagonal\+Vector\+Type\+\_\+ $>$}
Expression of a diagonal matrix. 


\begin{DoxyParams}{Parameters}
{\em Diagonal\+Vector\+Type\+\_\+} & the type of the vector of diagonal coefficients\\
\hline
\end{DoxyParams}
This class is an expression of a diagonal matrix, but not storing its own vector of diagonal coefficients, instead wrapping an existing vector expression. It is the return type of \mbox{\hyperlink{classEigen_1_1MatrixBase_a22df3e4cfe9e81535c3590fd91fd639c}{Matrix\+Base\+::as\+Diagonal()}} and most of the time this is the only way that it is used.
