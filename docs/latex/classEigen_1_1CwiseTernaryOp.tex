\subsubsection*{template$<$typename Ternary\+Op, typename Arg1\+Type, typename Arg2\+Type, typename Arg3\+Type$>$\newline
class Eigen\+::\+Cwise\+Ternary\+Op$<$ Ternary\+Op, Arg1\+Type, Arg2\+Type, Arg3\+Type $>$}
Generic expression where a coefficient-\/wise ternary operator is applied to two expressions.

This class represents an expression where a coefficient-\/wise ternary operator is applied to three expressions. It is the return type of ternary operators, by which we mean only those ternary operators where all three arguments are \mbox{\hyperlink{namespaceEigen}{Eigen}} expressions. For example, the return type of betainc(matrix1, matrix2, matrix3) is a \mbox{\hyperlink{classEigen_1_1CwiseTernaryOp}{Cwise\+Ternary\+Op}}.

Most of the time, this is the only way that it is used, so you typically don\textquotesingle{}t have to name \mbox{\hyperlink{classEigen_1_1CwiseTernaryOp}{Cwise\+Ternary\+Op}} types explicitly.
