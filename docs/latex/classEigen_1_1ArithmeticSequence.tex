\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename First\+Type, typename Size\+Type, typename Incr\+Type$>$\newline
class Eigen\+::\+Arithmetic\+Sequence$<$ First\+Type, Size\+Type, Incr\+Type $>$}
This class represents an arithmetic progression $ a_0, a_1, a_2, ..., a_{n-1}$ defined by its {\itshape first} value $ a_0 $, its {\itshape size} (aka length) {\itshape n}, and the {\itshape increment} (aka stride) that is equal to $ a_{i+1}-a_{i}$ for any {\itshape i}.

It is internally used as the return type of the Eigen\+::seq and \mbox{\hyperlink{namespaceEigen_a3a3c346d2a61d1e8e86e6fb4cf57fbda}{Eigen\+::seqN}} functions, and as the input arguments of Dense\+Base\+::operator()(const Row\+Indices\&, const Col\+Indices\&), and most of the time this is the only way it is used.

{\em Size\+Type} & type representing the size of the sequence, usually an Index or a compile time integral constant. Internally, it can also be a symbolic expression \\
\hline
{\em Incr\+Type} & type of the increment, can be a runtime Index, or a compile time integral constant (default is compile-\/time 1)\\
