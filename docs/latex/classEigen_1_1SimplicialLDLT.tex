class Eigen\+::\+Simplicial\+LDLT$<$ Matrix\+Type\+\_\+, Up\+Lo\+\_\+, Ordering\+\_\+ $>$}
A direct sparse \mbox{\hyperlink{classEigen_1_1LDLT}{LDLT}} Cholesky factorizations without square root. 

This class provides a LDL$^\wedge$T Cholesky factorizations without square root of sparse matrices that are selfadjoint and positive definite. The factorization allows for solving A.\+X = B where X and B can be either dense or sparse.

In order to reduce the fill-\/in, a symmetric permutation P is applied prior to the factorization such that the factorized matrix is P A P$^\wedge$-\/1.
